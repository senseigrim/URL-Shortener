
# URL Shortener

The URL Shortener is a python project that is used to store and shorten urls. You can either input a new url, find a old one, find all old ones, or delete all of them.

## Installation

Install the `urlshortener.exe` file from the repository. Simply run it and all the dependencies will be installed.

## Usage
After running the `urlshortener.exe` file a option menu will come up. Type the number next to the option you would like to select to select it.
